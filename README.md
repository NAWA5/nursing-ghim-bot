# Nursing Ghim Bot

This bot collects questions from a Telegram channel and writes them to a Google Sheet.

## Environment Variables
Create a `.env` file containing the following keys:

- `API_ID`
- `API_HASH`
- `BOT_TOKEN`
- `SHEET_URL`
- `CHANNEL_USERNAME`
- `GOOGLE_CREDENTIALS` â€“ path to your Google service account JSON file used to access the spreadsheet.
